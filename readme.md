# BrainLog

BrainLog is a personal knowledge management and draft writing system designed for authors. It helps you organize your thoughts, ideas, and drafts in a structured manner.

## Features

- **Note Organization**: Create directories and subdirectories to categorize your notes.
- **Rich Text Editing**: Write and format drafts with a focused typewriter interface.
- **Search Functionality**: Quickly find notes using keywords or tags.
- **Customizable**: Tailor the system to fit your workflow.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/jacobslabs/brainlog.git
    ```
2. Navigate to the project directory:
    ```bash
    cd brainlog
    ```
3. Install dependencies:
    ```bash
    npm install
    ```

## Usage

1. Start the application:
    ```bash
    npm run dev
    ```
2. Open your browser and navigate to `http://localhost:3000`.
